# emergenceModelR: Educational Simulations for Emergence and Complexity

`emergenceModelR` provides simplified computational simulations for
teaching and exploring emergence, self-organization, complexity,
cellular automata, agent interactions, network growth, and
information-based metrics.

## Details

These functions are educational abstractions. They do not fully model
real biological, social, ecological, cognitive, or conscious systems.
